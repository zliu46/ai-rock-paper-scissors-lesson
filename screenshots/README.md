# Screenshots Guide

This folder contains reference images for the AI Rock Paper Scissors lesson.

## GitHub Repository
https://github.com/zliu46/ai-rock-paper-scissors-lesson.git

## Purpose
These screenshots help teachers and students understand what each stage of the Teachable Machine workflow should look like. They are meant to support the lesson plan, not replace it.

## How to Use These Screenshots
Teachers can use these images to:
- preview the full workflow before class
- model each step during instruction
- help students troubleshoot when their screens look different
- confirm that students are at the correct stage of the process

## Caption Style
Use short, teacher-friendly captions under each image.

Examples:
- “Click Get Started to begin a new Teachable Machine project.”
- “Choose Image Project for hand gesture recognition.”
- “Use the Standard image model for this lesson.”
- “Create three classes: Rock, Paper, and Scissors.”
- “Add clear image samples for each hand gesture.”
- “Test the trained model in the preview window.”
- “Copy the exported model link for use in the coding project.”

## Best Practices
- Keep screenshots large enough to read.
- Crop out unnecessary browser clutter when possible.
- Use arrows or highlights only when they help attention.
- Keep the visual style consistent across all screenshots.
- Avoid including private account details or unrelated tabs.

## For Teachers
These screenshots are especially helpful for:
- first-time instructors using Teachable Machine
- substitute teachers or co-teachers
- students who need visual step-by-step support
