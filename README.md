# AI Rock Paper Scissors Lesson

A teacher-friendly classroom project that helps students use AI as a coding partner to build a simple Rock Paper Scissors game.

## Overview

In this lesson, students learn how to:
- write clearer prompts for an AI coding tool
- review AI-generated code critically
- debug with AI support
- reflect on the strengths and limits of AI-assisted coding

## What students will build

Students will create a simple Rock Paper Scissors game with basic gameplay logic, user interaction, and testing/debugging practice.

## Audience

Recommended for:
- middle school school students
- beginner to early-intermediate coders
- classrooms introducing AI-assisted programming

## Time

50–60 minutes, depending on student experience.

- `lesson-plan/` — lesson documents and teacher-facing materials

## Suggested teaching goals

By the end of the lesson, students should be able to:
- create a useful prompt for an AI tool
- evaluate whether AI-generated code matches their goal
- use AI to revise or debug code
- explain one benefit and one limitation of coding with AI
